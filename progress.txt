## Codebase Patterns
- Tauri v2 state management: wrap state in `Arc<T>` and use `.manage(state.clone())` then `State<'_, Arc<T>>` in commands
- SQLite in Rust: use `rusqlite` crate with `bundled` feature for cross-platform compilation
- Database path: use `~/.proxybot/` directory for all persistent data
- WAL mode: enable with `PRAGMA journal_mode=WAL;` for concurrent read/write
- Certificate storage: use PEM strings instead of Certificate objects for easier serialization and file I/O
- Interior mutability: when Tauri manages state as Arc<T>, use Mutex<T> inside T for fields that need mutation (e.g., regenerate_ca)
- macOS TUN interface: use `libc::socket(PF_SYSTEM, SOCK_DGRAM, SYSPROTOCOL_CONTROL)` to create utun control socket
- DNS database logging: clone domain before emitting Tauri event since params! macro takes ownership
- Tauri commands using State<'_, Arc<T>> need Arc<T> type in State parameter
