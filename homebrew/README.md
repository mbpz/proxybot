# Homebrew Tap for ProxyBot TUI

## Install

```bash
# One-time tap setup (if cloned to custom location)
brew tap mbpz/proxybot /path/to/proxybot/homebrew

# Or if published to GitHub Pages
brew tap mbpz/proxybot https://mbpz.github.io/proxybot

# Install
brew install proxybot-tui
```

## Usage

```bash
proxybot-tui
```

### Keyboard Shortcuts
- `q` / `Esc` - Quit
- `r` - Start proxy
- `s` - Stop proxy
- `c` - Clear request list
- `↑` / `↓` - Navigate request list
